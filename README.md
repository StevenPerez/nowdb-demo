# nowdb-demo
### Demo Using nowDB library

### Setup
Clone the repo using `git`:
```
    git clone git@github.com:StevenPerez/nowdb-demo.git
```

Inside the nowdb-demo folder, install the packages using `npm`:
```
    npm install
```

In the console / terminal run:
```
    npm run start
```

Open a couple of tabs in a Browser, go to http://localhost:4545/
